# Sales Data Analysis using SQL and Python

## Project Overview
This project analyzes sales data stored in a MySQL database using Python and SQL.

## Tools & Technologies
- MySQL
- Python
- Pandas
- SQLAlchemy
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- SQL JOINs between sales and product tables
- Revenue calculation
- Monthly revenue trend analysis
- Product-wise revenue analysis
- Data visualization using line and bar charts
